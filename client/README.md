# Coffee Valley - Client

Aplikasi frontend untuk sistem manajemen Coffee Valley.

## Tech Stack

- React 18
- Vite
- React Router v7
- Axios
- Tailwind CSS
- React Toastify

## Setup

1. Install dependencies:
```bash
npm install
```

2. Pastikan backend server sudah running di `http://localhost:3000`

3. Jalankan development server:
```bash
npm run dev
```

4. Buka browser di `http://localhost:5173`
